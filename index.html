System Modules

Command Documentation

Antinuke Protocol
&antinuke enable &antinuke disable

Enable or disable high-level security like antiban, antikick, and antiprune protection.

Whitelist
&wl @user

Grant immunity to trusted members against automated security checks.

IMPORTANT: Must whitelist from ALL events (ban, kick, etc.) for full safety.
Unwhitelist
&unwl @user

Removes immunity. Process is handled automatically across all security events.

AFK Status
&afk [reason]

Set an AFK status. Status is automatically cleared once you send a message.

Audio Engine
&play <query>

Search and play high-quality music directly in your voice channel.

NOTE: You must be in a voice channel (VC) to use this command.
Interactive Games
&games

Play interactive games with your community.

Utility Suite
&av @user &banner @user

View the full-size avatar or profile banner of a member.

USAGE: Mention a user to see theirs, or leave empty for self.
Information
&si &ui @user

Retrieve detailed information about the server or a specific user.

NOTE: If no user is mentioned with &ui, it displays your info.
Ticket System
&ticket setup

Create a professional support system with interactive buttons for your server.

GUIDANCE: Follow the step-by-step instructions provided by the bot.
Leveling
&level enable

Enables the experience and leveling system in the current channel.

NOTE: Run &level channel to set destination.
Welcome System
&greetsetup

Sets a custom welcome message for your server to greet new members.

NOTE: Run &greet channel #channel to specify destination.
Boost Tracking
&boost setup &boost message

Configure custom celebration messages for when members boost your server.

NOTE: Run &boost channel for destination setup.
AI Chatbot
&ai activate &ai deactivate

Enables or disables the interactive AI chatbot system.

USAGE: Run in the specific channel you want the AI to use.
Verification
&verification setup

Initialize a secure verification process to protect your server from bots.

GUIDANCE: Follow the setup wizard.
Counting Game
&counting enable

Enables a competitive counting minigame for your server members.

NOTE: Use &counting channel to specify the active channel.
Invite Tracking
&tracking enable

Tracks server activity including member invites and message counts.

TIP: Use &tracking leaderboard to see top contributors.
