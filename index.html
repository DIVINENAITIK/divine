Command Documentation

Master the power of Divine.

Antinuke Module
&antinuke enable &antinuke disable

Enable or disable high-level security like antiban, antikick, and antiprune.

Whitelist
&wl @user

Grant immunity to trusted members.

IMPORTANT: Must whitelist from ALL events (ban, kick, etc.) for full safety.
Unwhitelist
&unwl @user

Removes immunity. Process is handled automatically across all security events.

AFK System
&afk [reason]

Set an AFK status. Status is automatically cleared once you send a message.

Music Module
&play <query>

Search and play high-quality music directly in your voice channel.

NOTE: You must be in a voice channel (VC) to use this command.
Games Commands
&games

Play interactive games with your community.

blackjack chess tic-tac-toe country-guesser rps lights-out wordle 2048 memory-game number-slider battleship connect-four slots
Utility Module
&av @user &banner @user

View the full-size avatar or profile banner of a member.

USAGE: Mention a user to see theirs, or leave empty to see your own profile info.
Info Module
&si &ui @user

Retrieve detailed information about the server or a specific user.

NOTE: If no user is mentioned with &ui, it displays your own account information.
Ticket System
&ticket setup

Create a professional support system with interactive buttons for your server.

GUIDANCE: Run the command and follow the step-by-step instructions provided by the bot in your channel.
Leveling System
&level enable

Enables the experience and leveling system in the current channel.

NOTE: You must run &level channel and set a destination so level-up messages can be sent correctly.
Welcome System
&greetsetup

Sets a custom welcome message for your server to greet new members.

NOTE: You must run &greet channel #channel to specify where the welcome messages should be sent.
Boost System
&boost setup &boost message

Configure custom celebration messages for when members boost your server.

NOTE: You must run &boost channel to specify the destination for boost notifications.
AI Chatbot
&ai activate &ai deactivate

Enables or disables the interactive AI chatbot system in the current channel.

USAGE: Run the command in the specific channel where you want the AI to respond to messages.
Verification System
&verification setup

Initialize a secure verification process to protect your server from bots.

GUIDANCE: Run the command and follow the setup wizard to enable member verification.
Counting System
&counting enable

Enables a competitive counting minigame for your server members.

NOTE: Must use &counting channel to specify the channel where the system should be active.
Tracking System
&tracking enable

Tracks server activity including member invites and message counts.

TIP: Use &tracking leaderboard to see the top contributors in your server.
